Clash of code

Clash of code is a competition where your basic programming would be tested(C,C++).

I.In first round Basic MCQ's would be provided.

II.In second round some Intermediate level of MCQ's would be provided.

III. In last round 3 problem statement would be given.

Webber

Webber is a competition where your web knowledge would be tested.

I.In first round Basic MCQ's would be provided.

II.In second round some Intermediate level of MCQ's would be provided.

III. In last round context would be given and you have to build a website of that context.

HotKeys

In this competition you have to use only keyboard and restrict the use of mouse.

I.In first round Basic MCQ's would be provided.

II.In second round some task would be given that need to be completed with use of keyboard only.

Flash

Flash is a competition where your photography skills are tested.

I.Here we will provide you the list of items and you have to click the photos of the items in the list in the most beautiful way.
